üìä Marking Criteria

Your solution is evaluated across three key areas. We're lenient and focus on whether your code works!

Correct Output 50%

Does your code produce the right answer? This is what matters most! If your output matches the expected result, you'll get full marks for this section.

Code Works 30%

Does your code run without errors? Is the logic sound? Even if the output isn't perfect, working code with good logic earns points.

Good Approach 20%

Is your approach reasonable for this problem? Did you use appropriate data structures and algorithms? We don't nitpick style or efficiency.

üí° We're Lenient!

We focus on whether your code WORKS, not on style or efficiency. You won't lose points for variable names, missing comments, or unoptimized solutions. If it produces the right output, that's what counts!
